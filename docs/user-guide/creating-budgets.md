# Creating Budgets

Learn how to create and manage budgets in Centabit.

## Overview

Budgets help you plan your spending over a specific time period. Each budget can have multiple category allocations to track spending by category.

## Creating a New Budget

1. Navigate to the **Budgets** tab
2. Tap the **+** (plus) button
3. Fill in the budget details
4. Set category allocations
5. Save your budget

## Budget Details

### Name
Give your budget a descriptive name (e.g., "January 2024", "Monthly Budget", "Vacation Fund")

### Amount
Set the total budget amount for the period.

### Date Range
- **Start Date**: When the budget period begins
- **End Date**: When the budget period ends

Common periods:
- Monthly: 1st to end of month
- Bi-weekly: 14 days
- Custom: Any date range you need

## Category Allocations

Allocations divide your budget across different spending categories.

### Adding Allocations

1. In the budget form, scroll to "Category Allocations"
2. Tap "+ Add Category"
3. Select a category
4. Set the allocation amount
5. Repeat for all categories

### Tips

- Allocations should add up to <= your total budget
- Use categories that match your spending patterns
- You can adjust allocations later
