# Tracking Transactions

Record and organize your income and expenses.

## Adding a Transaction

1. Go to the **Transactions** tab
2. Tap the **+** button
3. Fill in transaction details
4. Save

## Transaction Fields

### Amount
The transaction amount (positive for expenses, negative for income).

### Category
Select a spending category to organize your transactions.

### Description (Optional)
Add notes about the transaction.

### Date
When the transaction occurred (defaults to today).

### Budget (Optional)
Assign to a specific budget to track against your plan.

## Search and Filter

Use the search bar to:
- Search by description
- Filter by date range
- Filter by category
- Filter by budget

## Tips

- Add transactions regularly for accurate tracking
- Use descriptive names
- Assign to budgets to see BAR metrics
- Use categories consistently
