# Categories

Create and manage spending categories to organize your transactions.

## Built-in Categories

Centabit comes with common categories:
- Groceries
- Restaurants / Dining
- Entertainment
- Transportation
- Shopping
- Utilities
- Health
- And more...

## Creating Custom Categories

1. Go to **Categories** tab (or Settings â†’ Categories)
2. Tap **+ Add Category**
3. Enter a name
4. Select an icon
5. Save

## Using Categories

### In Transactions
Assign a category to every transaction for better tracking.

### In Budgets
Allocate portions of your budget to different categories.

### In Reports
View spending breakdowns by category.

## Tips

- Use consistent category names
- Don't create too many categories (10-15 is ideal)
- Review and merge similar categories periodically
- Match categories to your actual spending patterns
