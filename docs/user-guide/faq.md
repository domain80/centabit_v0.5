# Frequently Asked Questions

Common questions about Centabit.

## General

### Is my data private?
Yes! All data is stored locally on your device. If cloud sync is added in the future, it would be optional and encrypted.

### Can I use Centabit on multiple devices?
Currently, data is stored locally on a single device. Multi-device support via cloud sync is under consideration based on user feedback.

### Does Centabit connect to my bank?
No. You manually enter transactions. This keeps your banking credentials private and secure.

## Features

### How does BAR work?
See [Understanding BAR](./understanding-bar.html) for a detailed explanation.

### Can I edit or delete transactions?
Yes, tap any transaction to edit or delete it.

### Can I have multiple budgets?
Yes! Create as many budgets as you need for different time periods or purposes.

### What's the difference between a budget and a category?
- **Budget**: A spending plan for a time period
- **Category**: A type of spending (groceries, entertainment, etc.)
- **Allocation**: How much of a budget is assigned to each category

## Troubleshooting

### My BAR seems wrong
- Check that all transactions are assigned to the correct budget
- Verify budget start/end dates are correct
- Make sure transaction dates are accurate

### Transactions aren't showing in reports
- Check the date filter range
- Verify transactions have categories assigned
- Try refreshing the app

### App is running slow
- Close and restart the app
- Check available storage space
- Update to the latest version
