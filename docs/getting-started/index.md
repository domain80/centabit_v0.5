## What is Centabit?

Centabit is a Flutter-based budgeting application with a local-first architecture, offline support, and beautiful Material 3 design.

### Key Features

- **Budget Management**: Create and track monthly budgets
- **Transaction Tracking**: Record and categorize expenses
- **BAR Metrics**: Budget Adherence Ratio for financial health
- **Offline-First**: Works without internet connection
- **Multi-Platform**: iOS, Android, Web, Desktop
